The **Commodities** (Merci) registry defines the types of goods that can be transported within the system.

## Key Features
- **Classification**: Categorize goods by type (e.g., Chemicals, Automotive, Consumer Goods).
- **Hazardous Materials**: Management of RID/ADR codes for dangerous goods.
- **Loading Requirements**: Specific handling instructions associated with commodity types.

## Technical Fields
- **Commodity Code**: Unique identifier for the goods type.
- **Description**: Detailed description of the commodity.
- **Hazard Class**: (Optional) Link to international safety standards.
