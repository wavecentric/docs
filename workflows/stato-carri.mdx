import { Callout } from 'nextra-theme-docs'

This feature offers two main functionalities:

1.  **View by**
2.  **Summary**

## 1. View by

This option allows you to visualize your train truck data by grouping it based on a specific attribute. The result is a bar chart that clearly displays the count for each group, making it easy to identify trends and patterns.

<Callout>
  **Description:** Generates a bar chart grouping train truck data by the selected attribute and displays the count per group.
</Callout>

### Input Field

To generate the bar chart, you must specify how the data should be grouped. The field marked with a red asterisk (*) is mandatory.

| Field Name                  | Description                                            | Required |
| --------------------------- | ------------------------------------------------------ | :------: |
| **Select field to group by**| Choose the attribute you want to use for grouping the data (e.g., status, location, type). |   Yes    |

After selecting the field, click the **Run** button to generate the visualization.

---

## 2. Summary

This function provides a high-level overview of all wagon data. It's perfect for when you need a quick, consolidated report without diving into specific details.

<Callout type="info">
  **Description:** Creates an executive summary of all wagon.
</Callout>

This action runs directly and does not require any additional input.
