---
title: "Public Quote Request"
description: "Standard web form submission for new logistics leads."
---

This workflow handles inquiries submitted via the "Get a Quote" public web form.

## Process Flow

<Steps>
  <Step title="Form Submission">
    User submits contact details and cargo requirements.
  </Step>
  <Step title="Identity Resolution">
    The system checks if the user exists.
    - **New User**: Created as `lead`.
    - **Existing User**: Profile updated (unless company is verified).
  </Step>
  <Step title="Automation Triggers">
    Two background processes are started:
    - **Send Quote Webhook**: Notifies the sales team (Activepieces).
    - **Cost Calc Webhook**: Estimates the shipment cost.
  </Step>
</Steps>
