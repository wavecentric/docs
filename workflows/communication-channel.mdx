### 1. Communication Channel

The Communication Channel is for interacting with clients and stakeholders directly from the AI LooP platform. It allows you to send service messages with attachments and retrieve the communication history for any given email address.


This module has two primary functions: **Send Service Message** and **View a Communication**.

---

#### Sending a Service Message

This feature allows you to send a user-defined service message via email, processed via SMTP. You can also include an optional attachment.

**How to Send a Message:**

1.  Ensure the **Send Service Message** tab is selected (it is the default view).
2.  **Recipient:** In the `Recipient *` field, enter the full email address of the person you want to contact. This field is mandatory.
3.  **Message:** In the `Message *` field, compose the body of your email. This field is also mandatory.
4.  **(Optional) Attach File:**
    *   To include an attachment, click the "Click to upload" area.
    *   Select the file from your computer.
    *   The file will be attached to the email.

5.  Click the **Run** button at the bottom to send your message.

<Callout type="info" title="Attachment Guidelines">
  You can upload one file per message. The supported file formats are **PDF, DOCX, PNG, JPG, and XLSX**. The maximum allowed file size is **5MB**.
</Callout>

---

#### Viewing Communication History

This feature allows you to look up the past conversation history associated with a specific email address, providing a complete context of all prior interactions handled through the platform.

**How to View History:**

1.  Select the **View a Communication** tab.
2.  An input field will appear (based on the description "Retrieves the history for a given email address").
3.  Enter the email address of the contact whose history you wish to review.
4.  Execute the search to retrieve and display the conversation thread.

<Callout type="tip">
  Use the "View a Communication" feature before contacting a client to quickly get up to speed on the latest conversations and provide more informed support.
</Callout>
